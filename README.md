# Web Development Fundamentals Project

This repository contains a collection of web development tasks completed as part of a multi‑stage university project.  
The goal of the project was to explore essential web technologies and understand how modern websites are structured and built.

## Project Overview

The project includes several independent components, each focusing on a specific technology or concept:

### 1. Git Version Control & XML Validation
- Used Git and GitHub Classroom for versioning.
- Worked with XML files and validated them using online validation tools.

### 2. HTML Page (Based on Provided Design)
- Recreated a static webpage following a given layout.
- Practiced basic HTML structure and formatting.

### 3. Semantic HTML Website
- Built a small website using semantic HTML5 elements.
- Improved accessibility and document structure.

### 4. Website Styled with CSS
- Added custom CSS styling.
- Practiced selectors, layout, colors, spacing, and typography.

### 5. JavaScript‑Controlled Webpage
- Added interactivity using JavaScript.
- Implemented DOM manipulation and event handling.

### 6. Simple HTTP Server
- Developed a basic HTTP server that displays directory contents.
- Practiced server‑side logic and file handling.

### 7. Single Page Application (SPA) with AJAX
- Built a small SPA using JavaScript and AJAX.
- Loaded content dynamically without reloading the page.
